## 齐天大圣的逍遥生活与新职任命

齐天大圣，一名声名显赫的猴子，官微财薄却广为人知。天宫小官伴其左右，生活悠哉，饱食终日，交友广泛，游历四方。他与三清称兄道弟，对四帝尊敬有加，与诸天神和谐共处。其生活无定所，四处游乐。

一日晨会，玉帝接见许旌阳仙人，仙人提及齐天大圣闲暇时广交天宫之友，无论官阶高低，皆成挚交，恐生未明之患，建议委以职务以防患未然。玉帝采纳建议，召见猴王，命其掌管蟠桃园，猴王欣然领命，前往探视。途中，偶遇土地公，告知其新职，继续前行至蟠桃园。
}

## 大圣游桃园

土地公公召唤种地、浇水、剪枝与扫地工人，一同向大圣鞠躬并陪同游览。大圣询问桃树总数，土地答曰：共计三千六百株。前部树果三千年熟，食之成仙且身手敏捷；中部树果六千年熟，食之长生不老；末部树果九千年熟，紫纹果实，食之寿比天地，光耀如日月。大圣闻之大悦，当日归去，仅赏树观亭。自此，每隔数日必至桃园游玩，不再外访交友。一日，大圣见一老桃树硕果累累，心生品尝之意，但碍于众神在场，不便妄动。遂命众人门外等候，佯称亭中小憩，众神退去。

## 大圣偷桃与仙桃盛宴

大圣脱衣帽，攀大树摘熟桃，悠然享用后，招呼众人归。数日后复偷桃，自得其乐。另一日，王母设宴，启宝阁，举办仙桃盛会。红衣、青衣等七仙女携篮前往仙桃园摘桃备宴。至园门，遇土地神、力士及齐天府官守护。仙女言奉王母命摘桃，土地神告以今年特殊情况，需齐天大圣许可方可入园，并言大圣因疲于园中亭内小憩。仙女促速召大圣，免误时，土地神遂偕仙女同往。
}

## 大圣失踪与仙女寻桃

在花亭，大圣(人名)不见踪影，仅留衣物，去向成谜。四处搜寻，一无所获。大圣悠然食桃，后化为四小人，于密叶下休憩。七仙女忧虑，言：“空手而归，如何是好？”旁神仙官慰藉：“既接使命，前行勿忧。大圣好游，或访友未归。可先摘桃，吾等代为禀报。”仙女依言，入林采摘。前树得二篮，中树添三篮。后树果稀，仅余数枚未熟，熟者已被猴王食之。终在南枝觅得半红半白桃一枚，青衣仙女压枝，红衣仙女摘取后，枝复原位。

## 大圣遇仙女摘桃

原来大圣变小于树上休憩，被嘈杂声唤醒。他即刻恢复原状，耳中取出金箍棒，怒斥窃桃者。七仙女慌忙跪地，说明为王母娘娘筹备‘仙桃会’而来，初至果园寻大圣未果，恐延误使命而先行采摘，恳求宽恕。大圣闻之怒气转喜，询宴请嘉宾。仙女细数：西天佛祖、菩萨等，南极观音，崇恩圣帝及诸岛仙人，北极玄灵，黄极黄角大仙，五方五老，皆列邀请之中。
}

## 仙桃大聚会的邀请与大圣的应对

无论宫殿内的神仙，从五星神仙至上八洞的三清、四帝，乃至中八洞的玉皇、九垒、山海神仙，以及下八洞的幽冥教主与人间仙人，全数参与了仙桃大聚会。大圣笑道："我能参加吗？"仙女回应："未闻有你。"大圣又言："怎不预留老孙一席？我乃齐天大圣。"仙女答："旧规如此，现如何安排，我们不知。"大圣随即施定身术，使仙女滞于桃树下，自己则乘彩云离园，往瑶池而去。途中偶遇赤脚仙人，大圣心生一计，问曰："仙长何往？"答曰："赴王母之邀，参加仙桃会。"大圣续言...

## 天宫邀请与瑶池奇遇

因筋斗云迅速，天帝托我先至天宫大殿邀众神朝拜，再共赴宴。大仙虽好奇，仍问："惯例瑶池礼毕方食，何故变之？"终妥协，乘彩云赴天宫。与此同时，大圣伪装大仙速往瑶池。抵宝阁，见布置华丽而神仙未至。四处探寻间，浓烈酒香扑鼻。右侧廊内，酿酒仙、拌料工、提水道人及烧火童正忙于清洗满载佳酿之坛。大圣心生馋意，碍于旁人，未敢妄动。
}

## 大圣的意外之旅与发现

几只小虫爬上面容，大圣迅速施法应对，使众人陷入短暂休眠。他趁机享受美餐，直至醉意朦胧，念及客将至，决意早寝。醉行间，大圣偏离路线，误入兜率天宫，心念一转，欲借此机会拜访太上老君。宫内寻觅，老君不在，只见丹炉火旺，旁有五葫芦金丹。大圣兴奋言：自修行起，深知内外兼修，梦寐以求金丹助人，今偶得此机，决心试之...

## 大圣盗丹与逃归花果山

大圣将葫芦内金丹尽数吞下，旋即意识到事态严重：'祸矣！若玉帝得知，吾命休矣！'他决定：逃！速返花果山，重做山大王。于是，大圣悄然离兜率宫，未循原路，而择一隐蔽路径疾穿西天门，乘云速归花果山。
}

只见旗子飘扬，武器闪闪发光，四个勇士正和七十二个妖怪头目努力练武。大圣(人名)大声说：'小子们，我回来了！'妖怪们一听，立刻放下武器跪下：'大圣(人名)，好久不见！怎么这么久才来？'大圣(人名)轻轻地说：'只是过了一点时间罢了。'说完，他就直接进了山洞深处。四个勇士打扫好大厅，行跪拜礼，众人问：'大圣(人名)在天宫干了什么？'大圣(人名)笑着说：'我觉得就半年，没想到人间已过了几年？'勇士解释道：'天上一天，人间一年。这次玉帝提升大圣(人名)为'天宫大圣(人名)'，负责两地安全，还派了神将保护。大圣(人名)开心地接受了，在齐天府任职。我因为空闲，被安排管理蟠桃园。近来听说王母举办'蟠桃宴'，没邀请我，就私自去了瑶池，尝了仙树果和仙酒。回来时，不小心走进太上老君(人名)的住处，又吃了他的五颗长生丹。担心玉帝察觉，就悄悄离开了天宫。'

## 大圣的仙酒派对

大圣的话语让众妖欢欣鼓舞，他们立刻准备美酒鲜果迎接，特地为大圣斟满椰子酒。大圣浅尝后皱眉拒绝，崩和芭勇士解释大圣惯于天庭佳酿，椰子酒不合其口味。大圣笑道：'你们如家人般亲切。今晨，我见瑶池走廊陈列上等瓶装酒，你们还未品尝。我再去取些，共饮之，愿友谊长存。'群猴闻言振奋。大圣随即离洞，施展神通，潜入瑶池宴。宴上众人皆沉睡，大圣轻取四坛佳酿，速返洞中，举办了一场欢乐无比的'仙酒派对'，共享美酒，笑声不断。
}
